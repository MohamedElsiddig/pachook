# pachook

pacman hooks administrator with disabled exemples

## Installation
With AUR :

    yaourt -S pachook
    
or :

    git clone https://github.com/papajoker/pachook.git
    cd pachook
    makepkg -si
    
## Usage

pachook {command} "baseHookName"
	 cat
	 edit
	 create
	 enable or start
	 disable or stop
